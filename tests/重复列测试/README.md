# 重复列测试

## 测试目的与范围
- 测试MultiMerge工具处理重复列的能力
- 验证工具在列名重复时的行为

## 数据来源与特点
- 包含重复列名的测试文件
- 测试工具如何处理列名冲突

## 执行步骤
1. 确保MultiMerge.exe已编译完成
2. 运行 `run_duplicate_columns_test.ps1` 脚本
3. 查看测试结果

## 覆盖场景
- 重复列名
- 不同数据的相同列名

## 预期结果
- 工具应能处理重复列名
- 输出文件应包含所有列

## 文件说明
- `input/`：测试输入文件
- `scripts/`：测试脚本
- `run_test.bat`：运行测试的批处理文件
