# 多列测试

## 测试目的与范围
- 测试MultiMerge工具处理多列数据的能力
- 验证多文件合并时列的对齐情况

## 数据来源与特点
- 三个数据文件，每个文件包含多个数据列
- 测试工具如何处理多列数据的合并

## 执行步骤
1. 确保MultiMerge.exe已编译完成
2. 运行 `run_test.bat` 脚本
3. 查看测试结果

## 覆盖场景
- 多列数据的合并
- 不同时间间隔的文件

## 预期结果
- 输出文件应包含所有文件的列
- 数据应正确对齐到时间戳

## 文件说明
- `input/`：测试输入文件
- `expected/`：预期输出文件
- `run_test.bat`：运行测试的批处理文件
