# 不相交数据测试

## 测试目的与范围
- 测试MultiMerge工具处理时间范围不相交的数据文件
- 验证插值方法在边界情况下的表现

## 数据来源与特点
- 两个数据文件，时间范围完全不重叠
- 测试工具如何处理这种情况

## 执行步骤
1. 确保MultiMerge.exe已编译完成
2. 运行 `run_test.bat` 脚本
3. 查看测试结果

## 覆盖场景
- 时间范围不相交的文件
- 不同插值方法的处理方式

## 预期结果
- 输出文件应包含两个文件的所有时间点
- 对于没有数据的时间点，应根据插值方法处理

## 文件说明
- `input/`：测试输入文件
- `expected/`：预期输出文件
- `run_test.bat`：运行测试的批处理文件
