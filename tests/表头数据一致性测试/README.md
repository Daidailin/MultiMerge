# 表头数据一致性测试

## 测试目的与范围
- 测试MultiMerge工具处理表头数据一致性的能力
- 验证工具在处理重复列名时的行为

## 数据来源与特点
- 包含相同列名的测试文件
- 测试工具如何处理列名冲突

## 执行步骤
1. 确保MultiMerge.exe已编译完成
2. 运行 `run_header_consistency_tests.ps1` 脚本
3. 查看测试结果

## 覆盖场景
- 相同列名的文件
- 不同数量列的文件

## 预期结果
- 工具应能处理列名冲突
- 输出文件应包含所有列
- 重复列名应有明确处理

## 文件说明
- `input/`：测试输入文件
- `expected/`：预期输出文件
- `scripts/`：测试脚本
- `run_test.bat`：运行测试的批处理文件
