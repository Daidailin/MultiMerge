# 混合模式测试

## 测试目的与范围
- 测试MultiMerge工具处理混合数据格式的能力
- 验证工具在复杂场景下的表现

## 数据来源与特点
- 包含不同时间间隔和数据密度的测试文件
- 测试工具的综合处理能力

## 执行步骤
1. 确保MultiMerge.exe已编译完成
2. 运行 `run_day4_mixed_test.ps1` 脚本
3. 查看测试结果

## 覆盖场景
- 混合时间间隔
- 不同数据密度
- 不同插值方法

## 预期结果
- 工具应能处理混合数据格式
- 输出文件应正确对齐所有数据

## 文件说明
- `input/`：测试输入文件
- `expected/`：预期输出文件
- `scripts/`：测试脚本
- `run_test.bat`：运行测试的批处理文件
