# 基本正确性测试

## 测试目的与范围
- 测试MultiMerge工具的基本功能正确性
- 验证不同插值方法的效果
- 确保工具能够正确处理各种输入文件格式

## 数据来源与特点
- 测试数据为模拟的传感器数据
- 包含不同时间间隔的数据点
- 覆盖了各种可能的输入情况

## 执行步骤
1. 确保MultiMerge.exe已编译完成
2. 运行 `run_basic_correctness_tests.ps1` 脚本
3. 查看测试结果与预期结果的对比

## 覆盖场景
- 不同插值方法（最近邻、线性插值、不插值）
- 不同数据密度的文件
- 时间戳对齐情况

## 预期结果
- 输出文件应包含正确的时间列
- 数据列应根据选择的插值方法正确对齐
- 无错误信息输出

## 文件说明
- `input/`：测试输入文件
- `expected/`：预期输出文件
- `scripts/`：测试脚本
- `run_test.bat`：快速运行测试的批处理文件
