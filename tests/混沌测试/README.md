# 混沌测试

## 测试目的与范围
- 测试MultiMerge工具处理不规则时间戳数据的能力
- 验证工具在复杂情况下的稳定性

## 数据来源与特点
- 包含不规则时间间隔的数据文件
- 测试工具的鲁棒性

## 执行步骤
1. 确保MultiMerge.exe已编译完成
2. 运行 `run_day4_chaos_tests.ps1` 脚本
3. 查看测试结果

## 覆盖场景
- 不规则时间间隔
- 不同插值方法的处理

## 预期结果
- 工具应能处理不规则时间戳
- 输出文件应包含所有时间点
- 数据应正确对齐

## 文件说明
- `input/`：测试输入文件
- `expected/`：预期输出文件
- `scripts/`：测试脚本
- `run_test.bat`：运行测试的批处理文件
