# 边界类型测试

## 测试目的与范围
- 测试MultiMerge工具处理边界情况的能力
- 验证工具在数据开始和结束时的行为

## 数据来源与特点
- 包含边界情况的测试文件
- 测试工具的边界处理能力

## 执行步骤
1. 确保MultiMerge.exe已编译完成
2. 运行 `run_test.ps1` 脚本
3. 查看测试结果

## 覆盖场景
- 数据开始边界
- 数据结束边界
- 不同插值方法的边界处理

## 预期结果
- 工具应能正确处理边界情况
- 输出文件应包含所有数据点

## 文件说明
- `input/`：测试输入文件
- `expected/`：预期输出文件
- `output/`：测试输出文件
- `run_test.bat`：运行测试的批处理文件
- `run_test.ps1`：运行测试的PowerShell脚本
