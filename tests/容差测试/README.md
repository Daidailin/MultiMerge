# 容差测试

## 测试目的与范围
- 测试MultiMerge工具处理时间容差的能力
- 验证工具在时间戳不完全匹配时的行为

## 数据来源与特点
- 包含时间戳略有差异的测试文件
- 测试不同容差设置的效果

## 执行步骤
1. 确保MultiMerge.exe已编译完成
2. 运行 `run_tolerance_tests.ps1` 脚本
3. 查看测试结果

## 覆盖场景
- 不同时间容差设置
- 时间戳不完全匹配的情况

## 预期结果
- 工具应能根据容差设置处理时间戳差异
- 输出文件应正确对齐数据

## 文件说明
- `input/`：测试输入文件
- `scripts/`：测试脚本
- `run_test.bat`：运行测试的批处理文件
