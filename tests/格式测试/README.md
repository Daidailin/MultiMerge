# 格式测试

## 测试目的与范围
- 测试MultiMerge工具处理不同时间格式的能力
- 验证工具对不同分隔符的支持

## 数据来源与特点
- 包含不同时间格式的测试文件
- 测试工具的时间解析能力

## 执行步骤
1. 确保MultiMerge.exe已编译完成
2. 运行 `run_format_tests.ps1` 脚本
3. 查看测试结果

## 覆盖场景
- 不同时间格式（点号分隔毫秒、冒号分隔毫秒）
- 不同精度（毫秒、微秒）

## 预期结果
- 工具应能正确解析各种时间格式
- 输出文件应包含正确的时间列

## 文件说明
- `input/`：测试输入文件
- `scripts/`：测试脚本
- `run_test.bat`：运行测试的批处理文件
